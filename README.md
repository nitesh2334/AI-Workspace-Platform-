# Cortex — AI Workspace Platform

A premium AI workspace platform inspired by Linear, Raycast, ChatGPT, and modern SaaS product design.

Built with a strong focus on:

* product engineering
* frontend architecture
* streaming AI UX
* premium interactions
* scalable UI systems

---

# Live Features

## Authentication

* Supabase authentication
* protected routes
* persistent sessions
* secure server/client auth handling

## AI Chat Workspace

* real-time streaming AI responses
* OpenRouter integration
* markdown rendering
* syntax-highlighted code blocks
* retry + stop generation actions
* copy response action
* typing indicators

## Conversation System

* persistent conversations
* save/load chat history
* rename conversations
* delete conversations
* create new chats
* conversation sidebar

## Model Switching

Supports multiple AI models through OpenRouter:

* GPT-4o Mini
* Claude 3.5 Sonnet
* Gemini Flash

## Premium Product Experience

* responsive dashboard UI
* command palette (Cmd/Ctrl + K)
* Framer Motion animations
* loading skeletons
* polished empty states
* mobile-friendly interactions
* keyboard-first UX

# Getting Started

## Clone Repository

```bash
git clone https://github.com/nitesh2334/AI-Workspace-Platform-.git
```

## Install Dependencies

```bash
npm install
```

## Environment Variables

Create:

```bash
.env.local
```

Add:

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
OPENROUTER_API_KEY=
```

## Run Development Server

```bash
npm run dev
```
